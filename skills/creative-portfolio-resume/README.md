# creative-portfolio-resume

Build resumes for creative roles that balance visual design with ATS compatibility — showcasing portfolios, creative work, and design sensibility without sacrificing machine readability.

## When to Use

- Writing a resume for a creative role (designer, UX researcher, art director, copywriter, photographer)
- Structuring a resume that prominently features portfolio links and creative projects
- Balancing visual appeal with ATS parsing requirements for creative industry applications
- Reformatting a technical resume for a design or creative portfolio context
- Highlighting creative achievements, awards, and recognition alongside work experience

## What is Included

- `SKILL.md` — Core workflow: creative role analysis, portfolio integration, ATS-safe formatting, and visual hierarchy guidance
- `evals/evals.json` — 3 realistic test cases with assertions for trigger accuracy and workflow correctness
- `evals/trigger-eval.json` — 20 queries (10 should-trigger / 10 should-not-trigger) for description optimization

## Typical Invocation

- "Help me build a resume for a UX designer role that includes my Dribbble portfolio"
- "Create a creative resume that balances visual design with ATS compatibility"
- "Write a resume for a senior art director position — I have a strong portfolio to highlight"
- "Build a resume for this graphic designer role that stands out visually"
