# ai-native-product

Build AI-native products with agency-control tradeoffs, calibration loops, and eval strategies. Use when building AI agents, LLM features, or products where AI handles user tasks autonomously. Part of the Modern Product Operating Model collection.

## When to use

- Strategic planning and execution.
- Product management workflows.
- Aligning teams on product architecture and delivery.

## What is included

- `SKILL.md` - Core workflow and instructions.
- `templates/` - Directory with supporting templates.

## Typical invocation

- "Use the ai-native-product skill to help me plan this."
- "Apply the ai-native-product framework to my project."
