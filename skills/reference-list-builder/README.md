# reference-list-builder

Format professional references properly and prepare complete reference materials — including reference sheets, reference briefing notes, and email outreach scripts for referees.

## When to Use

- Formatting a professional reference list for a job application
- Preparing briefing notes to send to references before they're contacted
- Writing scripts or email templates to ask contacts to be references
- Selecting the right references for a specific role or industry
- Creating tailored reference sheets that align with the job you're applying to
- Preparing references for an executive or senior leadership job search

## What is Included

- `SKILL.md` — Core workflow: reference selection, formatting, briefing note preparation, and outreach scripts
- `evals/evals.json` — 3 realistic test cases with assertions for trigger accuracy and workflow correctness
- `evals/trigger-eval.json` — 20 queries (10 should-trigger / 10 should-not-trigger) for description optimization

## Typical Invocation

- "Help me format my professional reference list for this application"
- "Write a briefing email to send to my references before they're contacted"
- "Create a reference sheet that aligns with the senior PM role I'm applying for"
- "Help me choose and prepare 3 references for this executive job search"
