# abx-strategy

Build Account-Based Everything (ABX) GTM strategies for complex B2B sales. Use when working on ABM strategy, ICP scoring, messaging architecture, product launches, or pipeline acceleration for companies with <500 accounts, $100K+ deals, and 6+ month sales cycles.

## When to use

- Strategic planning and execution.
- Product management workflows.
- Aligning teams on product architecture and delivery.

## What is included

- `SKILL.md` - Core workflow and instructions.
- `templates/` - Directory with supporting templates.

## Typical invocation

- "Use the abx-strategy skill to help me plan this."
- "Apply the abx-strategy framework to my project."
