# resume-quantifier

Find opportunities to add metrics and quantifiable results to resume bullets — including estimation techniques for when exact data is unavailable.

## When to Use

- Identifying which resume bullets need quantifiable metrics added
- Estimating plausible metrics when you don't remember exact numbers (team size, budget, percentage improvement)
- Converting vague impact statements ("improved performance") into credible quantified achievements ("reduced page load by 40%")
- Teaching yourself how to find and frame metrics from your work history
- Reviewing a resume to find every bullet that could benefit from a number

## What is Included

- `SKILL.md` — Core workflow: metric opportunity identification, estimation frameworks, quantification patterns, and credibility validation
- `evals/evals.json` — 3 realistic test cases with assertions for trigger accuracy and workflow correctness
- `evals/trigger-eval.json` — 20 queries (10 should-trigger / 10 should-not-trigger) for description optimization

## Typical Invocation

- "Find every opportunity to add metrics to my resume"
- "Help me estimate numbers for my resume bullets — I don't have exact data"
- "Quantify my resume achievements using reasonable estimations"
- "Add measurable results to these resume bullets based on industry benchmarks"
